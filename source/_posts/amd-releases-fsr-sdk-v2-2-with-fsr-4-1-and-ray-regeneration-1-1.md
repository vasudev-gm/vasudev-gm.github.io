### Quick Report

AMD has released FSR SDK 2.2, bundling updated Redstone-era features including FSR 4.1 and Ray Regeneration 1.1 for supported Radeon platforms. The update is positioned around clearer motion detail, improved reconstruction quality, and stronger denoising behavior in ray-traced scenes without requiring major pipeline rewrites.
<!-- more -->

The package also includes Frame Generation 4.0 and Radiance Caching in technical preview, with feature support tied to RDNA 4 class Radeon RX 9000 GPUs and DirectX 12 Shader Model 6.6 requirements for some components. AMD notes that Vulkan is not currently supported in SDK 2.2, leaving API coverage focused on the DirectX path for this release.
