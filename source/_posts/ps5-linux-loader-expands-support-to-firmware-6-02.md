### Quick Report

The ps5-linux-loader project has widened Linux support on PlayStation 5 consoles, now covering firmware versions up to 6.02. That still keeps the work in old firmware territory, but it gives modders a much broader set of PS5 systems to experiment with.<!-- more -->

The latest update also adds working drivers for the built-in WLAN chip, which removes one of the rough edges from running Linux on the console. The developer behind the project says newer firmware versions are not expected to get support, so the new compatibility ceiling is likely the practical end point.
