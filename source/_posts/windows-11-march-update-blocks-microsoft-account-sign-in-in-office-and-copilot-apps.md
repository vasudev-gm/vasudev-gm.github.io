### Quick Report

A confirmed Windows 11 March update issue is causing Microsoft account sign-in failures in several Microsoft apps, including Teams Free, OneDrive, Edge, Word, Excel, and Microsoft 365 Copilot. Affected systems are reported to be running KB5079473 on Windows 11 24H2 and 25H2, where apps can incorrectly report no internet connectivity during sign-in even when the connection is active.
<!-- more -->

Microsoft says restarting the device may temporarily clear the broken network state, but the condition can return and trigger repeated sign-in problems. The company is preparing a fix and has indicated that no Windows Server platforms are affected by this specific bug.
