### Quick Report

Microsoft has begun rolling out the optional KB5089573 update for Windows 11, adding its Low Latency Profile feature that temporarily raises CPU frequency during app launches. The company frames the change as a way to improve responsiveness for in-box applications and parts of the system interface.
<!-- more -->

Microsoft claims around 40% faster launch behavior for first-party apps such as Edge and Outlook, with some UI paths like Start and context menus seeing larger gains in specific cases. The update is being delivered through a controlled feature rollout, so behavior may not activate immediately on every system after installation.
