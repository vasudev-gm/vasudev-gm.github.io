### Quick Report

Microsoft has released PowerToys 0.99 with new desktop controls that mirror common Linux behavior, including Alt+left-click drag to move windows and Alt+right-click to resize from the nearest edge. The update also introduces Power Display, a taskbar flyout for changing monitor source, power state, brightness, contrast, and saved display profiles.
<!-- more -->

Other changes include command dock refinements, persistent calculator history in Command Palette, and expanded keyboard-layout editor mapping for non-physical keys and chord setup. While these additions are packaged in PowerToys rather than core Windows, they continue Microsoft\'s pattern of importing popular Linux and macOS productivity interaction models into optional Windows tooling.
