### Quick Report

Multiple users report that the March 2026 Windows 11 update KB5079473 is tied to severe stability problems, including frozen systems, random reboots, and in some cases a denied-access error on the C drive. Microsoft has acknowledged the C drive issue and says it is working on a fix, while broader user reports continue to track additional reliability problems after installation.
<!-- more -->

The current explanation from Microsoft points to Samsung Galaxy Connect as a key trigger for the C drive access denial on specific Samsung Galaxy Book 4 and desktop configurations running 24H2 or 25H2 builds. While official status notes currently scope the root cause around Samsung systems, community reports suggest not every affected user may fit a single hardware pattern, so the full impact surface is still being clarified.
