### Quick Report

Microsoft introduced the Surface RTX Spark Dev Box, a compact developer PC built around an NVIDIA RTX Spark superchip delivering up to 1 petaflop of AI compute and 128 GB of unified memory; it is designed to run large local models and speed prototyping.
<!-- more -->

The machine ships with Windows 11 Pro tuned for developers (WSL2 with GPU passthrough, VS Code, GitHub Copilot, PowerShell 7) and includes security features such as secured-core PC, BitLocker and Entra integration for enterprise management. Surface RTX Spark Dev Box will be sold later this year exclusively from Microsoft.com.
